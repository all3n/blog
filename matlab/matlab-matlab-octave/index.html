matlab & ocatve

  1. index 从1开始
  2. length 长度

io

  1. data = open(“data.file”);

cmd

  1. clear ; close all; clc
    1. clc：清除命令窗口的内容，对工作环境中的全部变量无任何影响
    2. close：关闭当前的Figure窗口
    3. close all:关闭所有的Figure窗口
    4. clear：清除工作空间的所有变量
    5. clear all：清除工作空间的所有变量，函数，和MEX文
  2. fprintf(‘hello world \n’)
  3. % comment

convension

  1. X (m,n + 1)
  2. y (m,1)
  3. theta (n+1,1)

matrix Utility

  1. 列用;分割 行用,分割
  2. a = [1;2;3]
  3. X = data(:,1) 取第一列
  4. X = data(:,[1,2]) 取1,2 列
  5. 获取非0序号 Find
    1. pos = find(y==1)
  6. size
    1. [m , n] = size(X)
  7. X = [ones(m, 1) X];
  8. 全0矩阵 zeros(x,y)
  9. 全1矩阵 ones(x,y)
  10. 单位矩阵 I = eye(n) Identity Martix
  11. 矩阵赋值：assignment
    1. X(i,y) = n

gen

  1. linspace (BASE, LIMIT, N)

  2. logspace(-2, 3, 5)

logic

  1. for

plot

  1. figure; % open a new figure window
  2. plot
  3. 3D:surf(X,Y,V)
  4. 等高线: contour(X,Y,V,logspace(-2,3,20))

minimizers

  1. options = optimset(‘GradObj’, ‘on’, ‘MaxIter’, 400);
  2. [theta, cost] = fminunc(@(t)(costFunction(t, X, y)), initial_theta, options);
