  1. shell实用快捷键
  • Ctrl + a 光标切换到行首（经常用）
  • Ctrl + e 光标切换到行尾
  • cd - 切换到上次目录
  • !! 上次命令（一般使用 针对忘记打sudo，下面打sudo !!即可）
  • Ctrl ＋l 清屏
  • Ctrl + f / Ctrl +b 等同左右箭頭
  • Ctrl + k 剪切光標后
  • Ctrl + u 剪切光標前
  • Ctrl + y 粘贴shell剪切板 （注意这三个命令和系统剪切板不共享）
