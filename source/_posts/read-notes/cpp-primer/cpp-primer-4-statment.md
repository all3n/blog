#  Statement 语句
## 简单语句
1. 空语句 ;
2. 声明语句
    1. int ival;
    2. 可以同时声明多个
        1. int ival =1,ival2 = 3;
    1. 建议按类型分组声明

3. IF
    建议if语句用括号括起来 避免歧义
    ```
     if( condition ){
          statement
     }else{ 
          statement
     }
    ```
4. switch
    1. 注意执行case 直到遇见break为止
5. for
6. while
